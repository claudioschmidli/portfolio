About Me

Who Am I?

Hi, I'm Claudio! I'm a Data Scientist and Natural Scientist working in the pharmaceutical industry, where I combine my passion for data-driven problem solving with my deep scientific background. Currently, I'm expanding my expertise through a Master's in Data Science while working as a software developer at Solvias, where I develop applications and conduct data analysis.

My journey into data science began during my PhD in a microscopy group, where I discovered the fascinating world of large-scale image data processing. This experience opened my eyes to the power of data analysis and set me on an exciting path combining scientific research with computational methods.

After my PhD, I joined Solvias's microscopy group, where I've continued to grow at the intersection of pharmaceutical research and data analysis. This unique combination of hands-on industry experience and academic pursuit allows me to bring fresh perspectives to complex challenges.

Below, you'll find a collection of my projects that demonstrate my growing expertise in data science and software development. These projects, primarily developed during my master's studies, showcase my analytical skills and programming capabilities. I'm always excited to connect with fellow data enthusiasts and explore new opportunities for collaboration. Feel free to reach out!

My Projects

Projects

Microflow Imaging Particle Classification

Neuronal Networks This project explores the application of deep learning models for classifying particle images obtained through Micro-Flow Imaging (MFI). It investigates the use of various pre-trained models, transfer learning, and fine-tuning techniques on a custom-collected MFI dataset. The findings demonstrate that deep learning techniques provide deeper and more meaningful insights from MFI data compared to traditional thresholding methods. Click here to visit the GitHub repository

Container Orchestration with Kubernetes

DevOps & Cloud This project involves deploying and managing the Yelb multi-tier web application in a cloud-native environment using Kubernetes. As part of the FHNW Cloud & Containerization course, it explores manual and automated deployment with Kubernetes manifests and Helm charts, emphasizing scalability, automation, and reproducibility. Integration with DigitalOcean Kubernetes enhances the deployment pipeline. Click here to visit the GitHub repository

Analysis of Switzerland's Railway Network

Graph Analysis This project analyzes the Swiss Federal Railways (SBB) network to uncover insights into traffic patterns, route utilization, and key nodes. Using data from the SBB API, it examines freight and passenger traffic over three years, including COVID-19's impact. Advanced graph analysis identifies critical nodes via Betweenness Centrality, calculates route lengths, and visualizes load distribution across railway lines. Click here to visit the GitHub repository

Predict Income

Machine Learning on Tabular Data This project, part of the Advanced Machine Learning module in the MAS Data Science program at the University of Applied Sciences Northwestern Switzerland (FHNW), focuses on predicting whether an individual's income exceeds $50K/year using the "Adult Income" dataset. Through techniques such as feature engineering, class imbalance handling, and model optimization, the study evaluates logistic regression, LightGBM, and AutoML models, highlighting the strengths of gradient boosting for tabular data. Click here to visit the GitHub repository

Mortality Prediction in Switzerland

Time Series Analysis This project, part of the Time Series Analysis module in the MAS Data Science program at the University of Applied Sciences Northwestern Switzerland (FHNW), analyzes weekly mortality data in Switzerland from 2010 to 2023. Using ARIMA modeling, it explores trends, seasonal patterns, and stationarity to forecast future mortality rates, contributing to insights for public health planning and resource allocation. Click here to visit the GitHub repository

Penguin Species Predictor

Software Engineering This project, developed as part of the Software Engineering for Data Scientists course at FHNW (University of Applied Sciences and Arts Northwestern Switzerland), highlights the integration of machine learning with professional software engineering practices. The focus was to create a reliable, maintainable, and deployable machine learning solution using a binary logistic regression model to predict penguin species based on physical characteristics like culmen length and depth. Click here to visit the GitHub repository

Miniaturized Protein Preparation Module

Robotics in Biology This PhD project at the University of Basel's C-CINA introduces a novel protein purification method using magnetic nanoparticles, an electromagnetic trap, and a microfluidic system to streamline cryo-EM sample preparation. I developed the magnetic trap's hardware and software for robotic integration, advancing engineering and biochemistry to enhance structural biology workflows.
